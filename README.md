# CivicBrain AI

CivicBrain AI is an AI-powered civic issue management platform that enables citizens to report community problems using text, images, and location data.

Features:
- AI issue categorization
- Image upload support
- Severity prediction
- Priority scoring
- Department assignment
- Interactive location mapping
- Issue tracking timeline
- Analytics dashboard

Technology Stack:
- Flask
- HTML/CSS/JavaScript
- Google Gemini API
- Chart.js
- Leaflet Maps
